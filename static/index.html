How It Works

Built for review sessions between engineering and management.

  • The left and right views show the original scene materials.
  • The center view highlights additions, removals, and modifications.
  • Camera movement is synchronized, so orbiting one panel updates all three.
  • Geometry, transform, and material changes are surfaced in the audit list.
Continuous version review for CAD should feel more like Git than file archaeology.
